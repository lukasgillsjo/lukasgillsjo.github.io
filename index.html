Projects

Raytracer in C++

Raytracer written in C++. Uses a method where no additional rays are created at intersections with objects. Instead every ray uses weighted randomization when hitting objects to decide how and in which direction to scatter. Mulltiple rays are sent from the camera through every pixel position to make up for this. Only handles spherical and plane primitives. Supports diffuse surfaces, dielectrics, metals, moving objects, depth of field, textures, and rectangular light sources.

Software rasterizer in C++

A simple software rasterizer. I started this project to better understand how rasterization engines really worked. Very much a work in progress :)

Raytracer in C#

My first simple raytracer. Every ray will continue to propagate in this manner until either the importance of the ray is below a certain threshold or the maximum number of iterations has been reached for that specific ray.
Supports refraction and reflection. Propagates and launches new rays at intersection points until the importance of the ray reaches a certain threshold or until the maximum number of intersections for that ray has been reached. Launches shadow rays in a grid pattern for direct lighting and calculates indirect lighting by using Monte Carlo integration; launching a number of new rays in a stochastic manner and then adding up the result of these rays weighted with a probability distribution.
